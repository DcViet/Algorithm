# Algorithm

"Thuật toán hình chữ nhật tối đại" (Maximal Rectangle Algorithm) : Thuật toán này áp dụng cơ chế quy hoạch động để tìm kiếm hình chữ nhật lớn nhất.
- Ý tưởng chính của thuật toán là xem mỗi hàng trong ma trận như một đường đơn giản và áp dụng thuật toán "[Largest Rectangle in Histogram](https://github.com/DcViet/Algorithm/blob/main/assets/Largest_Histogram.md)" trên các hàng để tìm kiếm hình chữ nhật lớn nhất.
